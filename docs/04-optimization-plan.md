# Optimization Plan (First 30 Days)

What we look at after the agent has been live for about 30 days, and what we’d change.

## What to analyze

**Conversion by path**
- % of conversations that end in AE booked (green) vs nurture (yellow) vs disqualified (red). If green is too low, we’re either too strict or the traffic mix is off. If red is tiny, we might be nurturing people we should cut.

**Drop-off points**
- Where do people stop replying? (e.g. right after Budget, or after the first question.) Tells us which questions feel heavy or which steps are too long. We’d look at “last message sent” and “no reply after” by conversation.

**Objection handling**
- How often do we get “I want to talk to someone” or “not ready” or “don’t want to share budget”? Which of those get escalated vs handled in flow? We want to see if we’re over-escalating or under-escalating.

**Time to outcome**
- For green leads: time from first touch to “AE booked.” For yellow: time to first nurture step. If it’s too slow, we might be asking too many questions or waiting too long to ask for the meeting.

**Quality of handoffs**
- When we do escalate, do those turn into meetings? If not, we might be escalating too early or passing bad context. Spot-check a sample of escalated threads.

**Nurture performance**
- Do any yellow leads later turn into green (e.g. reply to nurture and then book)? If almost none do, either our nurture content is wrong or we’re labeling too many as yellow.

## What we’d improve

- **Tighten or loosen criteria**  
  If we’re booking too many bad-fit calls, make green a bit stricter (e.g. require timeline in quarter). If we’re leaving money on the table, loosen one dimension (e.g. “exploring” timeline can still be green if BAN are strong).

- **Rewrite the questions that cause drop-off**  
  If everyone leaves at Budget, test a softer ask (“Do you already have budget for wellbeing, or is that something you’d need to get approved?”) or move it later in the flow.

- **Adjust escalation triggers**  
  If we’re sending too much to humans, add a single “anything else I can help with?” before escalating. If we’re under-escalating, add keywords or a “talk to someone” button earlier.

- **Shorten the flow**  
  If we have 8 messages before we ever ask for a meeting, test 4–5. Fewer steps often means more completes and more green leads captured.

- **Nurture cadence and content**  
  If yellows never convert, try different subject lines, fewer emails, or a different first piece of content. Test one variable at a time.

- **Revisit red definition**  
  If a lot of “red” leads later buy or book (e.g. from another channel), we’re disqualifying too aggressively. We’d relax the red rule for that case and maybe move it to yellow.

We’d run this as a 30-day review: pull the numbers above, read a sample of conversations, then ship 1–2 changes and measure the next 30 days.
